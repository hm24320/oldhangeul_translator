열 달이면 회태(=뱃속에서) 나오니, 효자의 아들은 두 손을 맞추어 나오되, 어미를 해하지 않게 나고, 오역의 아들은 어미의 배를 헤치며 어미의 간을 잡으며, 어미의 뼈도 박차니, 어미의 배에 칼로 찔러 기는 듯이 앓게 하니, 이 몸이 날 때에 열 가지 은혜다.￭ 참고현대의학에서 태아의 발생에 대하여 어떻게 설명하는가를 참고로 보임. ‘카네기 (태아) 발생의 기준과 배자 및 주요 기관의 발달, 태아기 중의 발생’ 중에서 초록함.1) 1~4주 : 크기(mm) : 4~5, 일수(日數) : 28.내용 : 원래 편평(偏平)했던 배자(胚子) 원반이 원통형으로 되고 신경관의 형성, 인두궁(咽頭宮), 눈, 귀, 손, 발의 발생태(發生態)가 되고, 23일경 박동을 시작한다.2) 5~8주 : 크기(mm) : 6~31, 일수(日數) : 56.5, 머리 둔부 길이 27~31mm.내용 : 상지(上肢)는 길어지고 손가락이 분명해지며 차차 정중(正中) 쪽으로 가까워지고, 하지(下肢)도 발가락과 무릎이 구별되며, 이 또한 정중 쪽으로 가까워진다.3) 9~12주 : 머리 둔부 길이(mm) : 35~93, 발 길이(mm) : 7~14, 체중(g) : 3~40.내용 : 9주부터 태아(胎兒)라 하는데, 분명히 인간과 비슷하다. 이 시기에 머리의 성장이 둔화되면서 신체의 부분들이 차차 성인과 비슷해지는데, 목이 차차 펴지면서 잘 구별되며, 얼굴 부위가 넓어지면서 바깥쪽에 자리했던 눈이 차차 복부 쪽으로 이동된다. 12주 말에는 태아가 피부의 자극에 반응하며, 부갑상선 호르몬이 생산되며, 혈액은 응고될 수 있고, 꼬리는 퇴행 변화하여 둔부로 흡수되어 완전히 없어진다.4) 13~16주 : 머리 둔부 길이(mm) : 95~140, 발 길이(mm) : 20~27, 체중(g) : 50~150.내용 : 이 시기에 얼굴이 더 인간답게 되며, 태아에 따른 차이가 나타나기 시작한다. 성장이 매우 빠르며, 특히 키가 커진다. 하악골이 발달해 아래턱이 관찰되고, 머리칼의 형(pattern)이 결정되고, 하지도 상당히 길어진다. 16주 초에는 산모의 복수 X선 사진에서 쉽게 관찰된다.5) 17-20주 : 머리 둔부 길이(mm) : 140~190, 발 길이(mm) : 33~39, 체중 : 200 ~350g.내용 : 이 시기에 성장은 둔화되지만 머리 둔부 길이는 약 50mm가 증가하며, 대개 산모는 태아의 운동을 쉽게 느낀다. 18주에 여아의 경우, 자궁은 완전히 형성되며, 20주에는 남아의 경우, 고환은 아직 후복부 벽에 위치하고, 태모가 나타나서 차차 신체의 모든 표면을 덮는다. 하지는 상당히 길어지고 신체의 다른 부위에 대한 비율이 성인에 유사해진다.6) 21~25주 : 머리 둔부 길이(mm) : 200~240, 발 길이(mm) : 45~49, 체중 : 450 ~1000g.내용 : 이 시기에는 체중이 뚜렷하게 증가한다. 신체는 아직 가늘지만 점차 균형이 잡히고, 피부는 빨리 성장하지만 피하조직이 적기 때문에 특징적으로 주름이 잡혀 있으며 매우 투명하기 때문에 모세혈관 속의 혈액이 보여 피부가 분홍 내지 적색을 띤다.7) 26~29주 : 머리 둔부 길이(mm) : 250~280, 발 길이(mm) : 50~62, 체중 : 1100 ~1600g.내용 : 태아의 폐는 공기를 들여 마실 수 있고, 폐혈관은 기체 교환이 가능할 정도로 발달해 있어서 이 시기에 태어난 미숙아는 살 수 있다. 또한 규칙적인 호흡운동을 하고 체온을 조절할 수 있을 정도로 중추신경 계통이 성숙해 있다.8) 30~34주 : 머리 둔부 길이(mm) : 290~340, 발 길이(mm) : 63~68, 체중(g) : 1800~2800.내용 : 발생 30주에 빛에 대한 동공반사가 있다. 34주경이 되면 피부는 부드럽고 분홍색을 띤다. 얼굴에는 태모가 없어지지만, 두모는 상당히 길다. 태지(胎脂)는 잘 발달되었고 손톱은 손가락 끝에 닿았다.(‘태지’는 자궁내의 태아 피부에 묻어 있는 기름기, 지방으로서 흰색으로 마치 돼지기름 굳은 모양 같은 지방 덩어리로 분만되는 태아의 피부에서 흔히 볼 수 있음.)9) 35~38주 : 머리 둔부 길이(mm) : 340~360, 발 길이(mm) : 69~83, 체중 : 3000~3400g.내용 : 발생 35주의 태아는 물체를 꽉 움켜쥘 수 있으며, 빛에 대해 자연스레 방향을 잡는다. 이 시기는 완성기로서 대부분의 태아는 토실토실하다. 머리는 태아기의 초기보다 신체의 나머지 부분에 비해 훨씬 작아졌지만, 둘레는 아직도 태아에서 가장 크기 때문에 이 머리 둘레는 산도를 통과할 때 중요하다. 36주에 머리와 복부 둘레는 같으나, 이후에는 복부 둘레가 크다. 이 시기에 대체로 남아가 여아보다 더 빨리 자라서 출생 시 남아가 더 무겁게 된다. (이후 분만)* 참고 문헌 : 『인체발생학』 3판(박형우, 군자출판사, 110~125쪽). 이 부분은 옹서(翁壻)의 인연으로 해서 도움을 준, 최호순 한양대학교 의과대학 교수의 수고가 있었음을 적어 둔다.
